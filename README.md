# CNAB Colorido

Página para visualizar arquivos CNAB240 e CNAB400 com a posições demarcadas por cores.

Inclui descrição dos campos

## Demo

[https://marcelo-lourenco.github.io/cnab/](https://marcelo-lourenco.github.io/cnab/)
